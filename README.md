### Backend


1. Go to the API folder

```bash
cd users-fullstack-project
cd api
```

2. Install dependencies
```bash
npm install
```

3. Run build
```bash
npm run build
```

4. Run prisma migrate
```bash
npm run db:migrate
```

3. Run Backend
```bash
npm run dev
```

### You can

4. Run prisma studio to see all registers
```bash
npm run db:studio
```
